# 🔐 Premium Password Generator

A modern Python-based Password Generator with a professional desktop interface built using Tkinter.

---

## 📌 Features

- Generate secure random passwords
- Choose password length
- Include:
  - Uppercase Letters
  - Lowercase Letters
  - Numbers
  - Symbols
- Copy password to clipboard
- Modern Black & Blue UI
- Uses Python's `secrets` module for secure password generation
