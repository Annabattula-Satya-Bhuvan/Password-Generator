# 🔐 Password Generator

A simple and responsive **Password Generator** built using **HTML, CSS, JavaScript, and Python**.

This project generates strong random passwords containing uppercase letters, lowercase letters, numbers, and special symbols. It was created as part of a Python programming task and includes a live web version that can be hosted using GitHub Pages.

---

## 📌 Project Overview

The Password Generator allows users to:

* Enter the desired password length
* Generate a secure random password
* Include letters, numbers, and special characters
* Copy the generated password to the clipboard
* Use the application directly in a web browser

---

## 🚀 Features

* Strong random password generation
* User-defined password length
* Clean and responsive user interface
* One-click copy to clipboard
* Simple and easy to use
